# 安装说明

## 安装步骤

### 1. 准备工作
确保您的Chrome浏览器版本为88或更高版本。

### 2. 加载扩展
1. 打开Chrome浏览器
2. 在地址栏输入 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择本项目文件夹（包含manifest.json的文件夹）
6. 点击"选择文件夹"

### 3. 验证安装
- 在Chrome工具栏中应该能看到扩展图标
- 点击图标应该能打开侧边栏或弹窗

## 首次配置

### 1. 获取Coze API密钥
1. 访问 [Coze官网](https://www.coze.cn/)
2. 注册并登录账号
3. 创建工作流用于生成封面图
4. 获取个人API令牌和工作流ID

### 2. 配置扩展
1. 点击扩展图标打开侧边栏
2. 在"API配置"区域输入：
   - Coze个人令牌
   - 工作流ID
3. 点击"保存配置"

## 使用方法

### 1. 基本使用流程
1. 打开支持的公众号编辑器页面
2. 点击扩展图标打开侧边栏
3. 点击"自动识别标题"获取文章标题
4. 点击"生成封面图"等待AI处理
5. 预览生成的封面图
6. 点击"下载图片"保存到本地

### 2. 支持的编辑器
- 微信公众平台编辑器 (mp.weixin.qq.com)
- 秀米编辑器 (xiumi.us)
- 135编辑器 (135editor.com)
- 其他主流编辑器（通过通用选择器）

### 3. 快捷键
- `Ctrl/Cmd + Enter`: 生成封面图
- `Ctrl/Cmd + R`: 识别标题

## 故障排除

### 常见问题

**Q: 扩展图标点击后没有反应？**
A: 
- 检查是否在支持的页面中使用
- 尝试刷新页面后重试
- 检查浏览器是否支持侧边栏功能

**Q: 无法识别文章标题？**
A:
- 确保在支持的编辑器页面中
- 检查页面是否完全加载
- 尝试手动输入标题

**Q: API调用失败？**
A:
- 检查网络连接
- 验证API密钥是否正确
- 确认工作流ID是否有效
- 检查Coze账户余额

**Q: 图片下载失败？**
A:
- 检查浏览器下载权限
- 确认生成的图片URL有效
- 尝试右键保存图片

### 调试模式
1. 按F12打开开发者工具
2. 查看Console标签页的错误信息
3. 检查Network标签页的网络请求

## 卸载扩展

1. 打开 `chrome://extensions/`
2. 找到"公众号封面图生成器"
3. 点击"移除"按钮
4. 确认卸载

## 技术支持

如果遇到问题，请：
1. 查看浏览器控制台的错误信息
2. 检查扩展是否为最新版本
3. 确认配置信息是否正确

## 注意事项

- API密钥将安全保存在本地，不会上传到服务器
- 生成图片需要消耗Coze账户额度
- 建议在稳定的网络环境下使用
- 生成时间通常为10-30秒，请耐心等待
