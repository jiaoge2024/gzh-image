# 帮我开发一个浏览器插件，可以实现自动生成公众号封面图
1. 点击插件图标，可以自动移动到浏览器的右边
2. 点击按钮，插件可以自动识别公众号编辑器中的文章标题
3. 依据文章标题，调用coze工作流生成图片并返回图片预览

4. 生成图片的过程是通过调用 Coze工作流API实现的
API调用方法：
curl -X POST 'https://api.coze.cn/v1/workflow/run' \
-H "Authorization: Bearer 这里输入个人令牌" \
-H "Content-Type: application/json" \
-d '{
  "workflow_id": "这里输入coze_ID",
  "parameters": {
    "input": "这里输入的是文章标题"
  }
}'

input参数输入识别的文章标题

  1. 个人令牌 与coze_ID 这两个参数需要放在前端，由用户输入自己的秘钥后填充到工作流调用方法中，再运行。
  2. 首次需要用户输入自己的秘钥，然后保存到本地，下次使用的时候，直接从本地读取秘钥，不需要用户再次输入
  
5. 返回生成的图片链接，可以预览图片效果，并提供图片下载功能。
