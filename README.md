# 产品说明书：公众号封面图生成器

## 1. 产品简介

本产品是一款浏览器插件，旨在帮助微信公众号运营者快速、方便地生成符合规范的封面图。用户可以通过简单的操作，自定义标题、背景图片等元素，一键生成美观的封面图，并直接用于公众号文章。

## 2. 核心功能

*   **模板选择**：内置多种封面图模板，满足不同风格需求。
*   **自定义标题**：支持输入主、副标题，并调整字体、颜色、大小。
*   **背景更换**：允许用户上传自己的图片作为背景，或使用内置的精选图片。
*   **实时预览**：所有修改都能实时在预览窗口看到效果。
*   **一键下载**：生成最终的封面图并下载到本地。

## 3. 安装与使用

### 安装方法
1.  下载插件文件包（`.zip`格式）。
2.  解压文件。
3.  打开你的浏览器（如 Chrome、Edge 等）。
4.  进入“扩展程序”管理页面。
5.  打开“开发者模式”。
6.  点击“加载已解压的扩展程序”，选择刚刚解压的文件夹。
7.  安装完成，插件图标会出现在浏览器右上角。

### 使用流程
1.  点击浏览器右上角的插件图标，打开操作界面。
2.  在弹出窗口或侧边栏中，根据提示输入文章标题。
3.  选择或上传你喜欢的背景图片。
4.  调整各项参数，直到对预览效果满意。
5.  点击“生成图片”或“下载”按钮，即可保存封面图。

## 4. 技术栈

本插件主要使用以下技术构建：
*   HTML
*   CSS
*   JavaScript

## 5. 文件结构说明

*   `manifest.json`: 插件的配置文件，定义了插件的名称、版本、权限等。
*   `popup.html` / `popup.js`: 插件点击图标后弹出的窗口界面和逻辑。
*   `sidepanel.html` / `sidepanel.js` / `sidepanel.css`: 插件侧边栏的界面、逻辑和样式。
*   `background.js`: 插件的后台脚本，处理后台任务。
*   `content.js`: 内容脚本，用于在特定页面内执行操作。
*   `icons/`: 存放插件所需的各种尺寸的图标。
